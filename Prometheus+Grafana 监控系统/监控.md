## 怎么监控？
### 监控工具
- free
- vmstat
- df
- top

### 监控系统
- zabbix
- Prometheus

## 要监控什么？
- 硬件监控（温度、硬件故障）
- 系统监控（处理器、内存、硬盘）
- 应用监控（Nginx Tomcat PHP MySQL）
- 日志监控（系统日志 服务日志 访问日志 错误日志）
- 安全监控（WAF 敏感文件监控）
- API 监控（可用性 接口请求 响应时间）
- 业务监控（订单量 用户注册数 活跃用户数 推广效果）
- 流量分析（用户地理位置 页面访问情况 页面停留时间）

## 准备工作
- 熟悉被监控的对象
- 整理监控指标
- 告警阈值定义
- 故障处理流程

 prometheus 不擅长日志监控
